# vue-game

## Project 

这是面试公司给的面试题，制作一款`画圈`游戏。

- 要求：
0. 必须使用 `Vue模版+组件` 来完成，不可以使用纯Javascript实现。
1. 参考视频制作，界面不要求完全一样，但功能尽可能完善。
2. 依次显示下一步该哪个玩家，`X`或`O`
3. 记录步骤，并显示
4. 可返回到之前的某一步再继续
5. 同一个符号垂直和水平三个一样的一条直线，就是最后的胜利者


[游戏页面](https://github.com/rayeric/vue-game/public/images/game.png)

## Project setup
```
npm install
```
